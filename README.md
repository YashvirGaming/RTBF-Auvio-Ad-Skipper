# RTBF-Auvio-Ad-Skipper

This extension automatically handles advertisement transitions on RTBF Auvio. It is designed to detect commercial breaks, adjusting the video playback rate to advance through them or moving to the live stream edge.

### Features:
* **Automatic Playback Adjustment:** Accelerates standard video-on-demand advertisement playback speed.
* **Live Stream Edge Jumps:** Monitors ad buffers during live streams to return the player to the live edge.
* **Dashboard Switch:** Includes an interface switch to toggle the extension active or disabled.
* **Local Metrics:** Tracks a running count of ad sequences encountered.
